What is this?

Startify is a free, open source website which allows you to plan a business/startup. It uses the idea of "spitting" your ideas onto the page to get everything out, and then carefully planning and re-wording the things you've stated in a short period of time.

What is in the source code and where can I find it?

The source code contains HTML and Javascript and uses the W3.CSS framework for styling. It also contains a few CSS lines to set the font to Source Sans Pro - Bold.
